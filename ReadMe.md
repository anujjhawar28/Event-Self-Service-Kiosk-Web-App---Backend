# Server

## Don't Forget to create /images folder in root folder in order to save images on server

```
/images
```

## Project setup

```
npm install
```

### Create .env file

```
PORT=
```

### Compiles and hot-reloads for development

```
npm run start
```

### Swagger Endpoint

```
http://localhost:<port>/
```

### API Endpoint

```
http://localhost:<port>/api/<>
```
